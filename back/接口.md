## 用户接口

**login**

```json
POST /login
说明: 用户登录,登录成功后会返回token

入参说明: 
email: 用户邮箱 string
password: 用户密码 string


-----------------
示例输入1:
POST /login
body:
{
    "email":"abcd",
    "password":"1234"
}
示例输出1:
{
    "code": 200,
    "data": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJleHAiOjE2OTA1MjQwNzIsInVpZCI6Mn0.wDiVmKb1F8FqiJ9u2CsEk9WnQ31Usv9wCdWfgvY4_W8"
}

示例输入2:
POST /login
body:
{
    "email":"abcd",
    "password":"12234"
}
示例输出2:
{
    "code": 201,
    "msg": "邮箱或密码错误,登录失败!"
}
```

注意,token的功能暂时没用

## 点位接口

**查询单个点位**

```json
GET /position
说明: 通过Uri传入id,查询单个点位的点赞数,点踩数以及图文数据。

入参说明: 
id: 点位id int


----------------------
示例输入1:
GET /position/1
示例输出1:
{
    "code": 200,
    "msg": "",
    "data": {
        "like": 0,
        "dislike": 0,
        "description": "testtest"
    }
}

示例输入2:
GET /position/355
示例输出2:
{
    "code": 201,
    "msg": "No such id"
}
```

**查询特点角色,特点地图,特点技能的点位**

```json
GET /positions/:mid/:hid/:skill
说明: 通过Uri传入mid(地图id),hid(角色id),skill(技能名称),查询出符合要求的全部点位

入参说明:
mid: 地图id int
hid: 角色id int
skill	技能名称 string



---------------
示例输入1:
GET /positions/1/1/c
示例输出1:
{
    "code": 200,
    "msg": "",
    "data": [
        {
           	"id":1,
            "sx": 254.123,
            "sy": 123.1324,
            "px": 1230.12,
            "py": 123.523,
        }
    ]
}


示例输入2:
GET /positions/2/3/c
示例输出2:
{
    "code": 200,
    "msg": "",
    "data": []
}
```

## 管理员接口

**添加地图**

```json
POST /map
说明: 通过request body中的JSON传入map需要的名称,缩略图地址以及完整图像地址从而创建地图


入参说明:
name: 地图名称 string
url: 地图完整图地址 string
avatar:地图缩略图地址 string


---------------
示例输入:
POST /map
body:
{
    "name":"map name1",
    "url":"test url1",
    "avatar":"test avatar1"
}
示例输出:
{
    "code": 200,
    "msg": ""
}
```

**添加点位**

```json
POST /position
说明: 通过request body中的JSON传入position需要的角色id等信息从而创建点位

入参说明:
hid: 角色id int
mid: 地图id int
skill:技能名称 string
sx: 角色站立点x float
sy: 角色站立点y float
px: 角色技能释放点(技能落点)x float
py: 角色技能释放点(技能落点)y float
description: 技能描述长文本
---------------
示例输入:
POST /position
body:
{
   "hid":1,
   "mid":1,
   "skill":"c",
   "sx":254.123,
   "sy":123.1324,
   "px":1230.12,
   "py":123.523,
   "description":"testtest"
}
示例输出:
{
    "code": 200,
    "msg": ""
}
```

## 文件

>   文件上传的服务器为:
>
>   http://172.16.0.62:8080

**文件上传**

```json
POST /upload  上传图片随后会返回图片path

入参说明:
file: 图片 multipart.FileHeader
---------------
示例输入:
POST /upload

示例输出:
{
    "data": {
        "url": "https://valtec.oss-cn-chengdu.aliyuncs.com/swim.png"
    },
    "errno": 0
}

示例输入(上传失败的):
{
    "errno": 1,
    "message": "上传失败,请稍后再试..."
}
```

**文件删除**

```json
DELETE /delete  通过文件名删除图片

入参说明:
filename: 图片url
---------------
示例输入:
DELETE /delete
{
   "url":"https://valtec.oss-cn-chengdu.aliyuncs.com/swim.png"
}
示例输出:
{
    "errno": 0,
    "message": "删除成功!"
}

示例输入(上传失败的):
{
   "errno": 1, 
   "message": "删除失败,请稍后重试!"
}
```
